def read():

Задача №2

Для второй задачи нам нужно получить длинный ID доски. Ранее мы пользовались коротким, но сейчас этого будет недостаточно. Для того чтобы получить длинный ID, нужно отправить запрос следующего вида:

response = requests.get(base_url.format('boards/' + board_id), params=auth_params).json()

где board_id — короткий ID доски, который у нас уже есть. В ответе вы получите некоторые поля доски, с которой вы работаете, среди них будет поле id — оно-то нам и нужно. Заменим в самом начале файла значение переменной board_id на полученное. Выглядеть это будет примерно так:

board_id = "5d8210d0fe6fe82b22ee533e"

Теперь перейдём к созданию колонок для задач. Итак, мы должны уметь создавать колонки как при помощи передачи команды create_column <NAME>, так и при передаче несуществующего названия колонки командам create и move. Для начала реализуем функцию создания колонок:

def create_column(column_name):  
    return requests.post(base_url.format('lists'), data={'name': column_name, 'idBoard': board_id, **auth_params}).json()

Помните, что board_id теперь содержит значение "длинного" ID нашей доски. Добавим обработку команды create_column в наш клиент:

if __name__ == "__main__":
    ...
    elif sys.argv[1] == 'create_column':
        create_column(sys.argv[2])
    ...

Добавим вызов этой функции в create и move в случае, когда переданной колонки не существует. Для начала нам нужно понять, что колонки не существует. Для этого реализуем функцию column_check, которая будет возвращать ID колонки, если колонка с таким именем существует, и None, если такой колонки нет:

def column_check(column_name):  
    column_id = None  
    column_data = requests.get(base_url.format('boards') + '/' + board_id + '/lists', params=auth_params).json()  
    for column in column_data:  
        if column['name'] == column_name:  
            column_id = column['id']  
            return column_id

Обратите внимание, что в функциях create и move есть передача ID колонки с нужным именем. Перед этой операцией нам нужно иметь на руках это значение. Модифицированная функция create:

def create(name, column_name):  
    column_id = column_check(column_name)  
    if column_id is None:  
        column_id = create_column(column_name)['id']  
  
    requests.post(base_url.format('cards'), data={'name': name, 'idList': column_id, **auth_params})
